input 设置 :before、:after 无效

原因：要设置“之前”、“之后”的内容，要求该元素本身是要可以插入 DOM 内容的。而 input、img、iframe 等元素是不能包含其他元素的，因此设置 :before、:after 无效

prop 修改属性与相关选择器

在 css 中使用属性选择器，当使用了 jQuery 的 prop 方法修改属性值后，选择器依然可以监测到。
